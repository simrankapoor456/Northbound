# Northbound

> Stop Applying. Start Building Connections.

Northbound is a relationship-first AI career platform that helps software engineers discover remarkable companies, connect with the right people, and build meaningful professional relationships that create opportunities.

Our goal is **not** to help users send more applications.

Our goal is to help users become the kind of engineers great teams are excited to build with.

---

# Project Status

🚧 Phase 0 — Foundation

We are intentionally designing the company before writing production code.

This repository is the single source of truth.

Always consult the relevant documentation before making architectural or product decisions.

---

# Philosophy

Northbound is built around one simple idea:

> Stop Applying.
> Start Building Connections.

We believe authentic relationships create opportunities that applications alone often cannot.

Every feature should reinforce that philosophy.

---

# Core Principles

- Quality over quantity.
- Relationships over transactions.
- Builders over applicants.
- Progress over perfection.
- One brick at a time.

Whenever there is uncertainty, prefer the solution that helps users build meaningful professional relationships.

# Engineering Guidelines

Favor:

- simple architecture
- modular design
- production-quality code
- maintainability
- readability

Challenge unnecessary complexity.

Recommend simpler alternatives whenever possible.

If a feature does not align with Northbound's mission, question it before implementing.
